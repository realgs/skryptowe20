Exchange rates and sales API

This API provides exchange rates of three currencies (USD, EUR, GBP) and data on total sales in two currencies - original (USD) and converted at the exchange rate of the National Bank of Poland (PLN).

Exchange rate data and sales data are from 2014-12-28 to 2016-12-28.

Date format used is YYYY-MM-DD, e.g. 2015-10-13.

Exchange rate data - property interpolated is set to true for days that had no value and took their value from the previous (or next, depending on the situation) day that had the exchange rate. False otherwise.

There are currently four endpoints you can use:

  • /exchange-rates/{currency}/{date}

    • Provides exchange rate of the selected currency on the selected date.

    For example: /exchange-rates/GBP/2015-10-13

  • /exchange-rates/{currency}/{start_date}/{end_date}

    • Provides exchange rates of the selected currency on the selected time period.

    For example: /exchange-rates/EUR/2015-10-13/2015-10-18

  • /sales/{date}

    • Provides sales data for the selected date.

    For example: /sales/2016-10-13

  • /sales/{start_date}/{end_date}

    • Provides sales data for the selected time period.

    For example: /sales/2016-10-13/2016-10-19
